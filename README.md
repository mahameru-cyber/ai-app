# AI-App

Project AI-App berbasis PHP. Repository ini sudah **bersih dari secret**. Semua credential sensitif disimpan di `.env` lokal masing-masing pengguna.

---

## 📂 Struktur Repo

- `assets/` → CSS & JS  
- `vendor/` → dependency PHP (tidak ikut di GitHub, dibuat lewat composer)  
- `login-google.php` / `login-google-callback.php` → login Google OAuth  
- `.gitignore` → mencegah file sensitif ikut commit  
- `.env.example` → template environment variables
